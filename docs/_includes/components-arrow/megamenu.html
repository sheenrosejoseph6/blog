Arrow Megamenu megamenu.js

Arrow's dropdown megamenu that turns into an off-canvas layout for mobile.

Example

For a full working example, visit the mega menu example page. The basic structure of the code is show below.

{% highlight html %}
Arrow Electronics
Arrow Electronics
  • Grow My Business
      • Solutions
      • ...
      • Settings
          ...
{% endhighlight %}

Usage

Add .ar-offcanvas-megamenu to the navbar for propert styling and try to follow the example above. The full page example can give you a better idea of how to manage this plugin.

Via data attributes

Add data-spy="megamenu" to the .ar-offcanvas-megamenu element.

Via JavaScript

{% highlight js %} $('.ar-offcanvas-megamenu').megamenu() {% endhighlight %}
