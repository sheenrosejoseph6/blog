Usage

Via data attributes

Add data-spy="rtable" to the .table-responsive wrapper to trigger the plugin.

Via JavaScript

Call the responsive table via Javascript:

{% highlight js %} $('.table-responsive').rtable() {% endhighlight %}

Options

Options can be passed via data attributes or JavaScript. For data attributes, append the option name to data-, as in data-backdrop="".

Name type default description
breakpoint string 768 Sets the window width at which the responsive styling kicks in.
