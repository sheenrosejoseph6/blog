Responsive images

Images in this framework are responsive-friendly by default. If you're using Google Maps, you might run into some display problems due to the use of max-width on images. Also, there may be a problem with the use of * { box-sizing: border-box; }. The following snippet should avoid all those problems.

{% highlight scss %} /* Fix Google Maps canvas * * Wrap your Google Maps embed in a `.google-map-canvas` to reset this framework's * global `box-sizing` changes. */ .google-map-canvas, .google-map-canvas * { .box-sizing(content-box); max-width: none !important; img { display: inline; } } {% endhighlight %}

For other cases where max-width is not desired on an image, or any other element for that matter, add the .no-max class.
