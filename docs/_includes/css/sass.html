Using SASS

This framework's CSS is built on SASS, a preprocessor with additional functionality like variables, mixins, and functions for compiling CSS. Those looking to use the source SASS files instead of our compiled CSS files can make use of the numerous variables and mixins we use throughout the framework.

Variables

Variables are used throughout the entire project as a way to centralize and share commonly used values like colors, spacing, or font stacks. Grid variables and mixins are covered within the Grid system section.

Colors

Easily make use of two color schemes: grayscale and semantic. Grayscale colors provide quick access to commonly used shades of black while semantic include various colors assigned to meaningful contextual values.

{% highlight scss %} $gray-darkest: shade($arrow-gray, 96%); //#050606 $gray-darker: shade($arrow-gray, 66%); //#323838 $gray-dark: shade($arrow-gray, 33%); //#636e6f $gray: $arrow-gray; //#95a5a6 $gray-light: tint($arrow-gray, 20%); //#aab7b7 $gray-lighter: tint($arrow-gray, 33%); //#b7c2c3 $gray-lightest: tint($arrow-gray, 66%); //#dae0e0 {% endhighlight %}
{% highlight scss %} $brand-primary: $arrow-blue; //#009ddc $brand-secondary: $gray; //#95a5a6 $brand-success: $arrow-green; //#54b948 $brand-info: $arrow-light-blue; //#72cdf4 $brand-warning: $arrow-gold; //#fdbb30 $brand-danger: $arrow-red; //#e31b23 {% endhighlight %}

Scaffolding

A handful of variables for quickly customizing key elements of your site's skeleton.

{% highlight scss %} // Scaffolding $text-color: $gray-darker; //#050606 $disabled-color: $gray-light; //#aab7b7 $border-color: $gray-lightest; //#dae0e0 $body-bg: #fff; //pure white $disabled-bg: $gray-lightest; //#dae0e0 {% endhighlight %}

Links

Easily style your links with the right color with only one value.

{% highlight scss %} // Variables $link-color: $brand-primary; $link-hover-color: shade($link-color, 25%); // Usage a { color: $link-color; text-decoration: none; &:hover { color: $link-color-hover; text-decoration: underline; } } {% endhighlight %}

Note that the $link-color-hover uses a function, another awesome tool from SASS, to automagically create the right hover color. You can use darken, lighten, saturate, and desaturate among others.

Typography

Easily set your type face, text size, leading, and more with a few quick variables. This framework makes use of these as well to provide easy typographic mixins.

Vendor mixins

Vendor mixins are mixins to help support multiple browsers by including all relevant vendor prefixes in your compiled CSS. Some of these mixins come from Bourbon, a SASS mixin library that is bundled with the Arrow framework. More information on Bourbon's mixins can be found here.

Rounded corners

Today all modern browsers support the non-prefixed border-radius property. As such, there is no border-radius mixin, but this framework does include shortcuts for quickly rounding two corners on a particular side of an object.

{% highlight scss %} @mixin border-top-radius($radius) { border-top-right-radius: $radius; border-top-left-radius: $radius; } @mixin border-right-radius($radius) { border-bottom-right-radius: $radius; border-top-right-radius: $radius; } @mixin border-bottom-radius($radius) { border-bottom-right-radius: $radius; border-bottom-left-radius: $radius; } @mixin border-left-radius($radius) { border-bottom-left-radius: $radius; border-top-left-radius: $radius; } {% endhighlight %}

Gradients

Easily turn any two colors into a background gradient. Get more advanced and set a direction, use three colors, or use a radial gradient. With a single mixin you get all the prefixed syntaxes you'll need.

{% highlight scss %} #gradient > .vertical(#333; #000); #gradient > .horizontal(#333; #000); #gradient > .radial(#333; #000); {% endhighlight %}

You can also specify the angle of a standard two-color, linear gradient:

{% highlight scss %} #gradient > .directional(#333; #000; 45deg); {% endhighlight %}

If you need a barber-stripe style gradient, that's easy, too. Just specify a single color and we'll overlay a translucent white stripe.

{% highlight scss %} #gradient > .striped(#333; 45deg); {% endhighlight %}

Up the ante and use three colors instead. Set the first color, the second color, the second color's color stop (a percentage value like 25%), and the third color with these mixins:

{% highlight scss %} #gradient > .vertical-three-colors(#777; #333; 25%; #000); #gradient > .horizontal-three-colors(#777; #333; 25%; #000); {% endhighlight %}

Heads up! Should you ever need to remove a gradient, be sure to remove any IE-specific filter you may have added. You can do that by using the reset-filter mixin alongside background-image: none;.

Utility mixins

Utility mixins are mixins that combine otherwise unrelated CSS properties to achieve a specific goal or task.

Clearfix

Forget adding class="clearfix" to any element and instead add the clearfix mixin where appropriate. Uses the micro clearfix from Nicolas Gallager.

{% highlight scss %} // Mixin @mixin clearfix() { &:before, &:after { content: " "; display: table; } } // Usage .container { @include clearfix(); } {% endhighlight %}

Horizontal centering

Quickly center any element within its parent. Requires width or max-width to be set.

{% highlight scss %} // Mixin @mixin center-block() { display: block; margin-left: auto; margin-right: auto; } // Usage .container { width: 48em; @include center-block(); } {% endhighlight %}

Sizing helpers

Specify the dimensions of an object more easily.

{% highlight scss %} // Mixins // For an object with equal widths and height, just enter one parameter: @mixin size($width $height) { // Output: width: $width; height: $height; } // Usage .image { .size(30em 24em); } .avatar { .size(3em); } {% endhighlight %}

Resizable textareas

Easily configure the resize options for any textarea, or any other element. Defaults to normal browser behavior (both).

{% highlight scss %} @mixin resizable($direction: both) { // Options: horizontal, vertical, both resize: $direction; // Safari fix overflow: auto; } {% endhighlight %}

Truncating text

Easily truncate text with an ellipsis with a single mixin.

{% highlight scss %} // Mixin @mixin ellipsis($max-width: 100%) { display: inline-block; max-width: $max-width; overflow: hidden; text-overflow: ellipsis; white-space: nowrap; } // Usage .branch-name { @include ellipsis(15em); } {% endhighlight %}

Retina images

Specify two image paths and the $1x image dimensions, and this framework will provide an $2x media query. If you have many images to serve, consider writing your retina image CSS manually in a single media query.

{% highlight scss %} .img-retina($file-1x; $file-2x; $width-1x; $height-1x) { background-image: url("${file-1x}"); $media only screen and (-webkit-min-device-pixel-ratio: 2), only screen and ( min--moz-device-pixel-ratio: 2), only screen and ( -o-min-device-pixel-ratio: 2/1), only screen and ( min-device-pixel-ratio: 2), only screen and ( min-resolution: 192dpi), only screen and ( min-resolution: 2dppx) { background-image: url("${file-2x}"); background-size: $width-1x $height-1x; } } // Usage .jumbotron { .img-retina("/img/bg-1x.png", "/img/bg-2x.png", 100px, 100px); } {% endhighlight %}
